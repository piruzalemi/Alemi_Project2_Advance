An Alemi Full-Stack Application @Columbia Univ. BootCamp - March 14, 2020

This interactive Full_stack Application Accesses a MongoDB/SQL, via Flask. Its front-end entailed a D3 + JavaScript + Leaflet + Bootstrap coding

Users can access: 1. State of 2020 Presidential Candidates 2. Latest status of earthquakes/ polys 3. Status of bicycles in NY
